# Validador de Senhas

Aplicação web simples desenvolvida com JavaScript para verificar a força de senhas em tempo real.

## Acesse o projeto
https://nodeexpress-81cr.onrender.com/

## Funcionalidades
- Verificação de força da senha (fraca, média, forte, muito forte)
- Barra de progresso dinâmica
- Feedback em tempo real
- Exibição da senha digitada
- Interface minimalista e responsiva

## Tecnologias utilizadas
- HTML5
- CSS3 (estilo minimalista, inspirado no padrão Apple)
- JavaScript (vanilla)

## Como funciona
A força da senha é calculada com base nos seguintes critérios:
- Mínimo de caracteres
- Presença de letras maiúsculas
- Uso de números
- Uso de caracteres especiais

Cada critério aumenta o nível de segurança, refletido na barra de progresso e na mensagem exibida.

## Responsividade
O projeto é totalmente responsivo, funcionando bem em dispositivos móveis e desktop.

## Objetivo
Projeto desenvolvido para fins acadêmicos, com foco na prática de:
- Manipulação do DOM
- Eventos em JavaScript
- Validação de dados no front-end
- Organização de interface
